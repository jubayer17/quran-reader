# Quran Reader

Static Quran reader built with Next.js, Tailwind CSS, and a local Quran dataset.

## Development

Run the client app:

```bash
npm run dev
```

## Scripts

- `npm run dev` starts the Next.js app.
- `npm run build` creates the production build.
- `npm run start` runs the production build.
- `npm run lint` checks the codebase.

## Notes

- Surah listing and surah detail pages are statically generated.
- Search runs server-side against the local dataset.
- Reader settings persist in localStorage.
